# E-COMMERCE_SALES_ANALYSIS-SQL-POWER_BI
End-to-End E-Commerce Sales Analytics project using PostgreSQL and Power BI. This project analyses sales performance, customer behaviour, product categories, marketing channels, customer satisfaction, and revenue trends using advanced SQL queries and interactive dashboards.

# 🛒 End-to-End E-Commerce Sales Analytics using PostgreSQL  & Power BI

## 📌 Project Overview

This project focuses on analyzing an E-Commerce sales dataset using PostgreSQL and Power BI to uncover meaningful business insights. The analysis covers sales performance, customer behavior, marketing effectiveness, product performance, shipping efficiency, and customer satisfaction.

The project demonstrates advanced SQL concepts such as Window Functions, CTEs, Views, Aggregate Functions, Ranking Functions, and analytical queries while creating an interactive Power BI dashboard for business reporting.

---

## 🎯 Project Objectives

- Analyze overall business performance.
- Identify top-performing countries and product categories.
- Evaluate customer purchasing behavior.
- Measure revenue trends over time.
- Analyze customer satisfaction and product ratings.
- Understand return patterns and delivery performance.
- Evaluate marketing acquisition channels.
- Create an executive dashboard for decision-making.
